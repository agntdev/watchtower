# Watchtower

Personal Telegram bot for private crypto watchlists, price-threshold and 1h percent-move alerts, on-demand prices, and optional morning summaries with owner analytics.

Spec: [`docs/blueprint.md`](docs/blueprint.md).

Built on [agnt-gm.ai](https://agnt-gm.ai). The whole bot is built and refined here as pull requests across successive build passes.
